# Розробка бота для керування персональними завданнями

Кваліфікаційна робота на підтвердження ступеня фахового молодшого
бакалавра ([ВСП «ППФК НТУ «ХПІ»](http://polytechnic.poltava.ua)). 
Керівник роботи – О.В. Бабич.

## Завдання до дипломної роботи:
Продукт має являти собою бот для платформи Telegram, який дозволяє керувати щоденними завданнями прямо в месенджері. Слід подбати про просту та зручну взаємодію з користувачем, реалізацію базових операцій з керування завданнями та стандартні поля завдань (заголовок, деталі, дата і час, вкладення чи посилання, відмітка завдання «зірочкою» тощо), збереження даних між сеансами, можливість експорту списку завдань в форматі Markdown.

## Використані технології:
- Python
- Pycharm
- Telegram
